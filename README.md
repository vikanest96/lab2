### Задание 1 : Входные и выходные данные

#### Данные на вход

На вход программа должна получать 1 целые число - нулевой элемент последовательности, и 1 натуральное число - количество элементов последовательности.

#### Данные на выход

На выход программа должна вывести следующие n элементов последовательности, данные элементы будут целыми числами так как a0 - целое.

### Задание 1 : Алгоритм

#### Алгоритм выполнения программы:

1. **Ввод данных:**
   
   Программа считывает 1 целое число и 1 натуральное число, обозначенные как `a0` и `n`.
   
3. **Создание массива**
   
   Программа создает массив размера n+1 и придает нулевому элементу массива значение a0.
   
4. **Вычисление каждого последующего элемента массива**
   
   Программа с помощью цикла, производит вычисление каждого последующего элемента, следущим образом:
   
   Если номер элемента i - четный, то значение данного элемента ai = a(i-1) - 1;
   
   Если номер элемента i - нечетный, то значение данного элемента ai = a(i-1) * 2;

5. **Вывод данных**
   
   Программа выводит значения элементов массива.

### Задание 1 : Программа

```java
import java.io.PrintStream;
import java.util.Scanner;

public class Main {
public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;

    public static void main(String[] args) {
        int a0 = in.nextInt();
        int n = in.nextInt();
        int[] a = new int[n + 1];
        a[0] = a0;

        for (int i = 1; i < n + 1; i++) {
            if (i % 2 != 0) {
                a[i] = a[i - 1] - 1;
            } else {
                a[i] = a[i - 1] * 2;
            }
        }
        for (int i = 1; i < n + 1; i++) {
            out.print(a[i] + " ");
        }
    }
}
```

### Задание 2 : Входные и выходные данные

#### Данные на вход

На вход программа должна получать 1 натуральое число - количество элементов последовательности, и n целых чисел - элементов последовательности.

#### Данные на выход

На выход программа должна вывести номер первого элемента подпоследовательности максимальной длинны, такой что ее можно сделать убывающей убрав 1 элемент, и номер элемента который требуется убрать.

### Задание 2 : Алгоритм

1. **Ввод данных:**
   Программа считывает натуральное число n - количество элементов последовательности, и сами элементы последовательности - n целых чисел.
2. **Длина максимальной убывающей подпоследовательности, начинающейся с индекса 1:**
   Чтобы найти длину максимальной убывающей подпоследовательности, начинающейся с индекса 1, "удаляем" элемент с индексом 0, то есть рассматриваем все элементы начиная с 1, до того момента пока подпоследовательность не перестанет быть убывающей.
3. **Длина максимальной убывающей подпоследовательности, заканчивающуюся элементом с индексом n-2:**
   Чтобы найти длину максимальной убывающей подпоследовательности, заканчивающуюся элементом с индексом n-2 , "удаляем" элемент с индексом n-1, то есть рассматриваем все элементы начиная с элемента индекса n-2 в обратном порядке, до того момента пока подпоследовательность не перестанет быть убывающей.
4. **Длина максимальной убывающей подпоследовательности, в которой удаленный элемент имеет индекс от 1 до n-2:**
   С помощью цикла перебираем все элементы от 1 до n-2.
   Для каждого элемента проверяем останется ли последовательность убывающей, если удалить выбранный элемент.
   Ищем длину максимальной убывающей подпоследовательности элементов стоящих слева от выбранного.
   Ищем длину максимальной убывающей подпоследовательности элементов стоящих справа от выбранного.
   Складываем получившиеся длины и получаем длину максимальной убывающей подпоследовательности, в которой удаленный элемент имеет индекс от 1 до n-2.
5. **Вывод данных:**
   Выводим длину максимальной убывающей подпоследовательности, индекс первого элемента данной подпоследовательности, индекс удаленного элемента.

### Задание 2 : Программа

```java
import java.io.PrintStream;
import java.util.Scanner;

public class Main {
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;

    public static void main(String[] args) {
        int n = in.nextInt();

        int[] a = new int[n];

        int max_d = 0;
        int I = -1;
        int Y = -1;
        int d = 1;


        for (int i = 0; i < n; i++) {
            a[i] = in.nextInt();
        }

        int i = 2;
        while ((i < n)&&(a[i-1] > a[i])) {    // Ищем длину максимальной убывающей подпоследовательности, начинающейся с индекса 1;
                d++;
                i++;
        }
        if (d > max_d) {
            max_d = d;
            I = 1;
            Y = 0;
        }

        d = 1;
        int j = n-3;
        while ((j >= 0)&&(a[j] > a[j+1])) {  // Ищем длину максимальной убывающей подпоследовательности, заканчивающуюся элементом с индексом n-2;
                d++;
                j--;
        }
        if (d > max_d) {
            max_d = d;
            I = n - 1 - d + 1;
            Y = n - 1;
        }

        for (int l = 1; l < n - 1; l++) {  // Ищем длину максимальной убывающей подпоследовательности, в которой удаленный элемент имеет индекс от 1 до n-2.
            if (a[l-1] > a[l+1]) {         // Проверяем останется ли последовательность убывающей если удалить выбранный элемент.

                int k = l-2;
                int d_l = 1;

                while ((k >= 0)&&(a[k] > a[k+1])) {
                    d_l++;
                    k--;
                }

                int m = l+2;
                int d_r = 1;

                while ((m < n)&&(a[m-1] > a[m])) {
                    d_r++;
                    m++;
                }
                d = d_l + d_r + 1;

                if (d > max_d) {
                    max_d = d;
                    I = l - d_l;
                    Y = l;
                }
            }
        }
        if (max_d != 1)
           out.println(max_d + " " + I + " " + Y);
        else
            out.print("NO");
    }
}
```

### Задание 3 : Входные и выходные данные

#### Данные на вход

Последовательность символов заканчивающихся точкой.

#### Данные на выход

На выход программа должна вывести "even" - если среди введенных символов четных цифр больше чем нечетных, "odd" - если нечетных больше чем четных, "=" - если их количества равны.

### Задание 3 : Алгоритм

1. **Ввод данных:**
   Программа должна получить на вход последовательность элементов, оканчивающаяся точкой.

2. **Цикл:**
   Перебираем все элементы последовательности.
   Проверяем является ли выбранный элемент четной или нечетной цифрой.
   Если является, увеличиваем соответсвующие переменные счетчика (add, even) на 1.
   
3. **Сравниваем числа:**
   Если add > even, выводим "add".
   Если add < even, выводим "even".
   Если add = even, выводим "=".

### Задание 3 : Программа
```java
import java.io.IOException;
import java.io.PrintStream;
import java.util.Scanner;

public class Main {
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;

    public static void main(String[] args) throws IOException {
       char a = (char)System.in.read();
       int add = 0;
       int even = 0;

       while ((int) a != '.') {
           if (((int) a >= '0')&&((int) a <= '9')) {
               if (((int) a - '0') % 2 == 0) {
                   even++;
               } else {
                   add++;
               }
           }
           a = (char)System.in.read();
       }
       if (add > even) {
           out.print("add");
       } else {
           if (add == even) {
               out.print("=");
           } else {
               out.print("even");
           }
       }
    }
}
```

### Задание 4 : Входные и выходные данные

#### Данные на вход

На вход программа должна получать натуральное число n - количество элементов массива, и n элементов массива.

#### Данные на выход

На выход программа должна вывести все элементы массива, которые больше среднего арифметического всех элементов стоящих справа от них.

### Задание 4 : Алгоритм

1. **Ввод данных:**
   Программа должна получить на вход число натуральное число n - количество элементов массива, и соответсвенно n элементов массива.

2. **Двойный цикл:**
   С помощью двойного цикла определяем для каждого элемента, сумму всех элементов стоящих правее.
   
3. **Ищем среднее арифметическое:**
   Далее определяем среднее арифметическое для этих элементов.

4. **Сравниваем числа:**
   Сравниваем элемент и среднее арифметическое всех элементов стоящих правее.
   
5. **Вывод данных:**
   Если среднее арифметическое меньше элемента, выводим его на экран.


### Задание 4 : Программа

```java
import java.io.PrintStream;
import java.util.Scanner;

public class Main {
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;

    public static void main(String[] args) {
        int n = in.nextInt();
        int[] a = new int[n];
        for (int i = 0; i < n; i++) {
            a[i] = in.nextInt();
        }
        for (int i = 0; i < n; i++) {
            int sum = 0;
            for (int j = i+1; j < n; j++) {
                sum+=a[j];
            }
            if ((double) a[i] > (double) sum / (n-i-1)) {
                out.print(a[i] + " ");
            }
        }
    }
}
```
