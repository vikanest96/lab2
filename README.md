### 1. Задание 1 : Входные и выходные данные

#### Данные на вход

На вход программа должна получать 1 целые число - нулевой элемент последовательности, и 1 натуральное число - количество элементов последовательности.

#### Данные на выход

На выход программа должна вывести следующие n элементов последовательности, данные элементы будут целыми числами так как a0 - целое.

### 2. Задание 1 : Алгоритм

#### Алгоритм выполнения программы:

1. **Ввод данных:**
   
   Программа считывает 1 целое число и 1 натуральное число, обозначенные как `a0` и `n`.
   
3. **Создание массива**
   
   Программа создает массив размера n+1 и придает нулевому элементу массива значение a0.
   
4. **Вычисление каждого последующего элемента массива**
   
   Программа с помощью цикла, производит вычисление каждого последующего элемента, следущим образом:
   
   Если номер элемента i - четный, то значение данного элемента ai = a(i-1) - 1;
   
   Если номер элемента i - нечетный, то значение данного элемента ai = a(i-1) * 2;

5. **Вывод данных**
   
   Программа выводит значения элементов массива.

### 3. Задание 1 : Программа

```java
import java.io.PrintStream;
import java.util.Scanner;

public class Main {
public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;

    public static void main(String[] args) {
        int a0 = in.nextInt();
        int n = in.nextInt();
        int[] a = new int[n + 1];
        a[0] = a0;

        for (int i = 1; i < n + 1; i++) {
            if (i % 2 != 0) {
                a[i] = a[i - 1] - 1;
            } else {
                a[i] = a[i - 1] * 2;
            }
        }
        for (int i = 1; i < n + 1; i++) {
            out.print(a[i] + " ");
        }
    }
}
```

### 4. Задание 2 : Входные и выходные данные

#### Данные на вход

На вход программа должна получать 1 натуральое число - количество элементов последовательности, и n целых чисел - элементов последовательности.

#### Данные на выход

На выход программа должна вывести номер первого элемента подпоследовательности максимальной длинны, такой что ее можно сделать убывающей убрав 1 элемент, и номер элемента который требуется убрать.

### 5. Задание 2 : Алгоритм

### 6. Задание 2 : Программа

```java
import java.io.PrintStream;
import java.util.Scanner;

public class Main {
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;

    public static void main(String[] args) {
        int n = in.nextInt();
        int[] a = new int[n];
        int max_d = 0;
        int I = -1;
        int Y = -1;
        int d = 0;
        int d_l = 0;
        int d_r = 0;


        for (int i = 0; i < n; i++) {
            a[i] = in.nextInt();
        }
        for (int j = 2; j < n; j++) {
            if (a[j-1] > a[j]) {
                d++;
            } else {
                break;
            }
        }
        if (d+1 > max_d) {
            max_d = d + 1;
            I = 1;
            Y = 0;
        }
        d = 0;
        for (int j = n-3; j >=0 ; j--) {
            if (a[j] > a[j+1]) {
                d++;
            } else {
                break;
            }
        }
        if (d+1 > max_d) {
            max_d = d + 1;
            I = (n-1) - max_d + 1;
            Y = n-1;
        }
        for (int i = 1; i < n-1; i++) {
            for (int j = i-2; j >= 0; j--) {
                if (a[j] > a[j+1]) {
                    d_l++;
                } else {
                    break;
                }
            }
            for (int j = i+2; j < n; j++) {
                if (a[j-1] > a[j]) {
                    d_r++;
                } else {
                    break;
                }
            }
            d = d_l + 1 + d_r + 1;
            if (d > max_d) {
                max_d = d;
                I = i - d_l - 1;
                Y = i;
            }
        }
        out.print(max_d + " " + I + " " + Y);
    }
}
```

### 9. Задание 3 : Программа
```java
import java.io.IOException;
import java.io.PrintStream;
import java.util.Scanner;

public class Main {
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;

    public static void main(String[] args) throws IOException {
       char a = (char)System.in.read();
       int add = 0;
       int even = 0;

       while ((int) a != '.') {
           if (((int) a >= '0')&&((int) a <= '9')) {
               if (((int) a - '0') % 2 == 0) {
                   even++;
               } else {
                   add++;
               }
           }
           a = (char)System.in.read();
       }
       if (add > even) {
           out.print("add");
       } else {
           if (add == even) {
               out.print("=");
           } else {
               out.print("even");
           }
       }
    }
}
```

### 10. Задание 4 : Входные и выходные данные

#### Данные на вход

На вход программа должна получать натуральное число n - количество элементов массива, и n элементов массива.

#### Данные на выход

На выход программа должна вывести все элементы массива, которые больше среднего арифметического всех элементов стоящих справа от них.

### 11. Задание 4 : Алгоритм

1. **Ввод данных:**
   Программа должна получить на вход число натуральное число n - количество элементов массива, и соответсвенно n элементов массива.

2. **Двойный цикл:**
   С помощью двойного цикла определяем для каждого элемента, сумму всех элементов стоящих правее.
   
3. **Ищем среднее арифметическое:**
   Далее определяем среднее арифметическое для этих элементов.

4. **Сравниваем числа:**
   Сравниваем элемент и среднее арифметическое всех элементов стоящих правее.
   
5. **Вывод данных:**
   Если среднее арифметическое меньше элемента, выводим его на экран.


### 12. Задание 4 : Программа

```java
import java.io.PrintStream;
import java.util.Scanner;

public class Main {
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;

    public static void main(String[] args) {
        int n = in.nextInt();
        int[] a = new int[n];
        for (int i = 0; i < n; i++) {
            a[i] = in.nextInt();
        }
        for (int i = 0; i < n; i++) {
            int sum = 0;
            for (int j = i+1; j < n; j++) {
                sum+=a[j];
            }
            if ((double) a[i] > (double) sum / (n-i-1)) {
                out.print(a[i] + " ");
            }
        }
    }
}
```
